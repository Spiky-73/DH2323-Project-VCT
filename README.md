# DH2323 Project: Voxel Cone Tracing

The goal of this project is to reimplement the Voxel Cone Tracing algorithm described in [this paper](https://www.diva-portal.org/smash/get/diva2:1886204/FULLTEXT01.pdf). The report of the project is available [here](DH2323-Project-report.pdf).

# Setup

This project uses CMake as its build tool. It can be configured by running the following command in the root directory:
```bash
cmake -B build
```

# Build and Run
To build the project, use the command:
```bash
cmake --build build
```

To run the generated executable you can run the command:
```bash
./build/DH2323Project
```
